Designed to make installing and operating FlexSlider 2 easier whithin WP.

== Installation ==

1. Upload `wp-flex-gallery.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go Settings -> WP Flex Gallery
